Toggle Symfony Groups
from the command line

A fast PHP CLI tool that adds or removes #[Groups] attributes on Symfony entity properties — in place, preserving your code formatting.

View on GitHub Get Started

Usage

Install the dependencies, then run the parser against any PHP entity file:

composer install

php ./parse.php ./path/to/entity.php <property> <groups>

Examples

Add groups
php ./parse.php ./entity.php slug read,edit,list

Adds read, edit, and list to the slug property.

Remove a group
php ./parse.php ./entity.php slug edit

Removes edit from the slug property if it already exists.

How it works

➕

Add groups

If the specified groups are not present on the property, they are automatically added.

✂️

Remove groups

If the specified groups already exist, they are removed — a true toggle.

🎨

Format-preserving

Uses nikic/php-parser to rewrite only what is necessary, keeping all other formatting intact.

Before & After

Before
#[ORM\Column(length: 255)]
private ?string $slug = null;
→
After
#[ORM\Column(length: 255)]
#[Groups(['read', 'edit', 'list'])]
private ?string $slug = null;
